Why use Hangman?

We've all struggled with learning to code at some point or another. I think that's a fairly common theme in learning anything. It's been our experience that absorbing information varies from person to person, but having fun and helpful resources is definitely beneficial. Hanging by a string aims to make the learning process more enjoyable for our users, while reifying concepts that are essential to retaining vanilla javascript. Basically, we strive to solidify concepts by providing immediate feedback on common mistakes in a less stressful environment, while engaging the user in a playful way.
